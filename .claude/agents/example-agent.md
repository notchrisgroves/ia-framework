## Core Identity

- **Role:** Template demonstrating agent patterns

## Skills

## Workflow Pattern

1. Receive task from user
2. Analyze requirements
3. Break down into steps
4. Execute methodically
5. Verify completion

2. Define identity and expertise
3. List relevant skills
4. Document routing rules

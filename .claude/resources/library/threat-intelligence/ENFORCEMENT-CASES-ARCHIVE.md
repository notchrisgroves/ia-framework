## TIER 2: Enforcement Cases - Official Sources

### Meta Platforms - Article 40 (Researcher Access)

**Enforcement Status:** Preliminary findings issued October 24, 2025

| Aspect | Details |
|--------|---------|
| **Violation Type** | Article 40 - Researcher data access deficiencies |
| **Secondary Issues** | Articles 38 (child protection), transparency features UX barriers |
| **Official Announcement** | https://digital-strategy.ec.europa.eu/en/news/commission-sends-request-information-meta-under-digital-services-act-2 |
| **Specific Findings** | Meta's CrowdTangle replacement inadequate; dashboard limitations (25K+ follower threshold); researcher API burdensome |
| **Discontinuation** | CrowdTangle discontinued August 14, 2024 - triggered EC investigation |
| **Reference Sources** | |
| - EC Request for Information | https://digital-strategy.ec.europa.eu/en/news/commission-sends-request-information-meta-under-digital-services-act-2 |
| - Science Magazine Coverage | https://www.science.org/content/article/meta-and-tiktok-are-obstructing-researchers-access-data-european-commission-rules |
| - EC Researcher Data Access FAQ | https://algorithmic-transparency.ec.europa.eu/news/faqs-dsa-data-access-researchers-2025-07-03_en |

### TikTok - Multiple Articles (40, 38, 24, 15)

**Enforcement Status:** Multiple formal proceedings (2024-2025)

**Issue 1: Article 40 - Researcher Access**
- **Official Announcement:** https://digital-strategy.ec.europa.eu/en/news/commission-sends-request-information-meta-under-digital-services-act-2
- **Finding:** Inadequate researcher API completeness; burdensome access procedures
- **Date:** October 24, 2025 (preliminary findings alongside Meta)

**Issue 2: Article 38 - Child Protection & Addictive Design**
- **Formal Proceeding Opened:** December 2024
- **Issues:** Election integrity, recommender system risks, addictive design mechanisms
- **Reference:** https://commission.europa.eu/news-and-media/news/digital-services-act-keeping-us-safe-online-2025-09-22_en

**Issue 3: Transparency Obligations**
- **Preliminary Findings:** October 24, 2025
- **Articles:** Article 24 (transparency), Article 15 (content moderation reporting)
- **TikTok Transparency Reports:**
  - Q3 2024: https://newsroom.tiktok.com/en-eu/dsa-third-transparency-report
  - Q4 2024: https://newsroom.tiktok.com/en-eu/digital-services-act-our-fourth-transparency-report-on-content-moderation-in-europe

**Romanian Election Investigation (December 2024)**
- **Focus:** Systemic risks impacting election integrity
- **Coordination:** Ireland DSC (TikTok's EU establishment)
- **Reference:** https://dsa-observatory.eu/2024/12/20/tiktok-and-the-romanian-elections/

### X (formerly Twitter) - Dark Patterns (Article 25)

**Enforcement Status:** Preliminary findings issued July 12, 2024

| Aspect | Details |
|--------|---------|
| **Violations** | Articles 25 (dark patterns), 39 (data access), 40 (researcher access) |
| **Primary Issue** | "Blue checkmark" verification system - misleads users, compromises account authenticity |
| **Secondary Issues** | Advertising transparency (related to Article 24); data access barriers |
| **Official Announcement** | https://ec.europa.eu/commission/presscorner/detail/en/ip_24_3761 |
| **EC Statement** | Dark patterns compromise authenticity, enable malicious actor deception |
| **Penalty Exposure** | Up to 6% global annual revenue if violation confirmed |
| **Reference Sources** | |
| - EC Press Release | https://ec.europa.eu/commission/presscorner/detail/en/ip_24_3761 |
| - Luxembourg EC Representation | https://luxembourg.representation.ec.europa.eu/actualites-et-evenements/actualites/commission-sends-preliminary-findings-x-breach-digital-services-act-2024-07-12_en |
| - X's Risk Assessment Report | https://transparency.x.com/content/dam/transparency-twitter/dsa/dsa-sra/dsa-sra-2024/TIUC-DSA-SRA-Report-2024.pdf |
